Methodology

📥 Data Collection

App Store reviews via Apple RSS feeds (up to 250/app/week). Trustpilot reviews via web scraping. Reddit posts via public JSON feeds. All automated weekly via GitHub Actions — zero API keys required.

🏷️ Classification

Dictionary-based keyword matching. Benefit terms: support, helpful, anxiety, loneliness, comfort, grief. Harm terms: addicted, manipulative, dependent, obsessed, unsafe, self-harm.

📐 Metrics

Benefit/Harm Rate: % of texts with ≥1 matching term. Net Sentiment: benefit_rate − harm_rate. Velocity: week-over-week net sentiment change.
